CORE VALUE

核心亮点

MSSP = XSIEM（看见）+ SOAR（行动）+ Ticketing（追踪），AI 与智能体深度融入真实运营链路。

亮点一：AI 智能化全面渗透

  • AI Chat 智能问答（威胁情报 / 工单查询 / 知识库）
  • AI 分析研判智能体，自动辅助告警研判
  • 工单查询封装为 MCP 服务，支持自然语言查询
  • 飞书告警 AI 摘要 + 聚合抑制信息展示
  • 邮件 .eml 附件 AI 解析分析
  • 外挂知识库支持私有文档导入 / 解析 / 向量化

亮点二：平台整合超越独立工具

  • XSIEM（检测）+ SOAR（编排）+ Ticketing（追踪）三合一
  • 工单天然携带告警上下文、处置历史
  • 告警自动收敛降噪 -> AI自动聚合 -> AI研判分析，减少人工处理
  • 飞书机器人巡检保证工单实时更新
  • XSIEM 2.0 平台升级，V2.2 接口全面适配
  • 49 个日志源 + 105 条告警规则覆盖全面
NEXT PHASE

下一阶段

多智能体协作完成工单闭环

  • 构建研判、处置、质检等智能体分工协作机制，围绕同一工单自动推进任务流转
  • 串联告警识别、证据收集、处置建议、结论沉淀等节点，形成可追踪的闭环链路
  • 在关键决策点保留人工确认，减少重复录入、跨系统搬运和人工跟进成本

外部情报联动实时威胁监测

  • 对接外部威胁情报源，持续更新 IP、域名、URL、文件哈希等 IOC 数据
  • 将外部情报与日志源实时碰撞，识别外部攻击、失陷主机和异常访问线索
  • 将命中结果自动回填至告警与工单，辅助研判优先级排序和处置路径推荐
RELEASE DETAILS

版本迭代明细

2025.12.17

V1.0 · 3 项交付
  • 工单 Link：支持多选工单建立一对一/一对多关联；详情页展示关联工单并可跳转；记录关联、解除、同步操作日志；联动 SOAR 展示 AI 研判分析；同状态告警合并聚合。
  • 工单附件表单：各事件类型匹配专属表单；不同工单类型预置带检查项的调查表单。
  • 工单 API 接口：开发工单创建、查询、修改、关闭开放接口；提供含参数、示例、错误码的完整 API 文档。

2025.12.30

V1.1 · 1 项交付
  • 工单详情：联动 SOAR 实现 AI 研判、告警聚合；多租户适配告警类型下拉，同步告警字典。

2026.01.30

V1.2 · 6 项交付
  • 工单批量导出：支持工单列表、详情页字段导出。
  • 工单双向关联：实现工单双向关联，关联范围限定本租户。
  • 工单搜索条件优化：详情页返回列表，保留原有搜索筛选条件。
  • 飞书通知优化：通知跳转链接替换为工单真实 URL。
  • 工单列表：新增告警类型展示，支持告警类型 DSL 查询。
  • 操作类优化：手动建单/专家分析设为非必填；告警名称改为工单名称，字符上限 200；新增复制粘贴快捷操作。

2026.02.28

V1.3 · 3 项交付
  • 邮件转成工单：指定发件人邮件自动生成工单，邮件内容附带 AI 分析。
  • 表单管理：表单字段支持拖拽调整；支持多标签页打开工单；限制专家分析录入长度；导出内容英文转中文。
  • 专家分析：支持单次批量上传多个附件。

2026.03.27

V2.0 · 6 项交付
  • 大屏功能：自定义大屏配置；同步工单数据，搭建运营类自定义大屏。
  • 工单详情：新增 AI 分析功能按钮。
  • 工单聚合：同源 IP、载荷的 TDP、WAF 外网攻击告警自动合并工单。
  • 情报接入：对接微步、VT 接口，判定外网 IP 并回填情报数据至工单。
  • 标签：抑制、关联工单列表增加标签展示。
  • 飞书告警：新增 AI 分析模块，展示聚合抑制相关信息。

2026.04.16

V2.1 · 2 项交付
  • 工单查询 MCP：工单查询功能封装为 MCP 服务。
  • XSIEM 2.0 升级：各子服务联调测试；系统版本间数据迁移；修复数据延迟、日志上传等异常；适配 V2.2 版本接口文档。

2026.04.28

V2.2 · 3 项交付
  • AI Chat：上线威胁情报、工单查询、知识库智能问答。
  • 外挂知识库：支持知识库文件上传，可在 AI 对话中调取。
  • AI 分析研判智能体：新增 AI 分析研判智能体能力。

2026.05.21

V2.3 · 7 项交付
  • 邮件转工单：支持 .eml 格式附件 AI 解析分析。
  • 专家分析：附件支持复制粘贴快捷上传；情况说明字数上限提升至 200 字。
  • 工单详情：对接腾讯情报接口，结构化可视化展示情报数据。
  • 区域配置：默认区域改为 NA，设为必选项，支持多区域选择。
  • API 接口文档-工单：优化接口描述内容；新增 AI 分析概要、全量分析、威胁情报接口文档。
  • 大屏管理账号：配置专属大屏管理账号。
  • 研判详情：更新告警关联分析页面展示字段。
